# [BOJ 7568 : 덩치](https://www.acmicpc.net/problem/7568)
- 랭크 : Silver 5
  <br><br>
  
### 문제 정리
- 자신보다 키, 몸무게가 둘 다 커야 덩치가 크다고 할 수 있다. 전체 사람에 대해서 덩치 등수를 출력하자.
   <br><br>

### 문제 풀이
- 접근 (Main) 
  - 나 자신을 1등으로 두고, 모든 배열에 대해서 비교를 진행한다. 나보다 덩치가 큰 사람이 있을 경우 rank를 올리게 되면, 비교 후 내 랭크를 알 수 있다.
