  <br><br>
  
### 문제 정리
- 재귀 횟수에 따른 챗봇의 응답을 출력한다.
<br><br>

### 문제 풀이
- 접근(Main) 
   - 매 호출 마다, ____를 미리 만들어두고 현재 depth의 모든 출력에 sb를 붙인다.
   - 이 때 depth가 원하는 재귀 횟수와 동일하면, 답변을 한 후 재귀를 마친다.
   - 여러 번 익숙해질 필요가 있다.
