# [BOJ 8911 : 거북이](https://www.acmicpc.net/problem/8911)
- 랭크 : Silver 2
  <br><br>
  
### 문제 정리
- 각 테스트 케이스에 대해서, 거북이가 이동한 영역을 모두 포함하는 가장 작은 직사각형의 넓이를 출력한다.

   <br><br>

### 문제 풀이
- 접근 (Main) 
    - 매 이동 때마다 점(x, y)의 최대 (maxX, maxY) 및 최소 (minX, minY)를 갱신한다.
    - 모든 수행 후, 최대 최소 점을 통해 직사각형 넓이를 구한다.
