# [BOJ 9613 : GCD 합](https://www.acmicpc.net/problem/9613)
- 랭크 : Silver 3
  <br><br>
  
### 문제 정리
- 테스트로 들어오는 수 여러 개에 대해서, 가능한 최대공약수 조합을 모두 구해서 합하는 문제이다.
   <br><br>

### 문제 풀이
- 접근 (Main) : 이중 for를 통해서 숫자의 조합을 구한다. gcd에 넘겨서 반환받은 최대공약수를 합에 더한다.
결과를 Long으로 받아서 출력해야 함에 유의한다.
