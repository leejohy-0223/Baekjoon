# [BOJ 1991 : 트리 순회](https://www.acmicpc.net/problem/1991)
- 랭크 : Silver 1
  <br><br>
  
### 문제 정리
- 첫째 줄에 전위 순회, 둘째 줄에 중위 순회, 셋째 줄에 후위 순회한 결과를 출력한다. 각 줄에 N개의 알파벳을 공백 없이 출력하면 된다.


<br><br>

### 문제 풀이
- 접근 (Main) 
  - 배열 또는 Node & Tree class를 통해 입력을 기반으로 트리를 만들고, 각각 탐색을 진행한다. 이해하고 외우자.
