# [BOJ 2563 : 색종이](https://www.acmicpc.net/problem/2563)
- 랭크 : Silver 5
  <br><br>
  
### 문제 정리
- 100 x 100의 도화지를 10 x 10의 색종이로 채우며, 채워진 총 영역을 구하는 문제이다.
<br><br>

### 문제 풀이
- 접근 (Main)
   - 100 x 100 배열을 만들고, 한 배열 칸을 1 넓이의 사각형으로 간주한다.
   - 입력된 색종이 시작점의 넓이에 맞게 boolean 배열을 true로 변경하고, count를 올려서 계산한다.
