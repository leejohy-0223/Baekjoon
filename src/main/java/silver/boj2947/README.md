# [BOJ 2947 : 나무 조각](https://www.acmicpc.net/problem/2947)
- 랭크 : 실버5
  <br><br>
  
### 문제 정리
- 처음 조각의 순서가 주어졌을 때, 위치를 바꿀 때 마다 조각의 순서를 출력하는 프로그램을 작성하시오.

   <br><br>

### 문제 풀이
- 접근 (Main) 
   - Bubble sort를 구현한다. 
   - swap이 발생할 때마다 값을 출력한다. 해당 while 반복에서 한번이라도 sorting이 일어났으면, 다시 반복하기 위해서 swap을 true로 check 한다.
