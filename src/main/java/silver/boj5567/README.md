# [BOJ 5567 : 결혼식](https://www.acmicpc.net/problem/5567)
- 랭크 : Silver 2
  <br><br>
  
### 문제 정리
- 상근이는 자신의 결혼식에 학교 동기 중 자신의 친구와 친구의 친구를 초대하기로 했다. 상근이의 동기는 모두 N명이고, 이 학생들의 학번은 모두 1부터 N까지이다. 상근이의 학번은 1이다.

- 상근이는 동기들의 친구 관계를 모두 조사한 리스트를 가지고 있다. 이 리스트를 바탕으로 결혼식에 초대할 사람의 수를 구하는 프로그램을 작성하시오.
   <br><br>

### 문제 풀이
- 접근 1 (Main) : ArrayList<ArrayList>에 양방향 연관관계를 작성하고, bfs로 2단계 까지만 조사해서 List에 넣는 방식으로 해결했다.
- 접근 2 (MainRef) : boolean 2차원을 만들어, 양방향 연관관계를 작성한다. 1과 연결된 노드 대상으로 다시 한번 for문을 돌려, 제 2의 boolean table에 표시 후 true인 것들의 개수를 count 한다.
