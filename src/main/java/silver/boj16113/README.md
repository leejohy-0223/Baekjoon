# [BOJ 16113 : 시그널](https://www.acmicpc.net/problem/16113)
- 랭크 : 실버2
  <br><br>
  
### 문제 정리
- 첫 번째 줄에 시그널을 해독하여 나오는 숫자들을 순서대로 출력한다.

<br><br>

### 문제 풀이
- 접근(Main) 
  - 맨 위의 모양을 통해 1, 4 여부를 판별한다. 1, 4는 DFS에서 개수를 통해서 처리한다. 
  - 나머지는 checkNumber 메서드에서 string으로 만든 후 비교해서 처리한다.
